Environnement de travail sous Linux
  • Editeur de texte

Vous pouvez utiliser l'un des éditeurs de texte disponibles : gedit, Nedit, kwrite, emacs, xemacs, kdevelop:c/c++,
Ces éditeurs disposent de la coloration syntaxique.

pour kwrite: Tools->highlighting->sources->C
pour kdevelop: Tools->highlighting->sources->C et Tools->Indentation->C style

Pour éditer un fichier, par exemple avec l'éditeur "kwrite" :
kwrite & ou kwrite nom-de-fichier &

  • Fichier noms.txt

Créer dans votre "home directory" (cd ou cd ~) un fichier qui s'appelle noms.txt et qui contient les noms
des 2 membres du binome de TD sous le format suivant: 

Jean-Jacques.Dupond@etu.utc.fr 

Alain.Durand@etu.utc.fr
  • Création du dossier td1

Créer un répertoire "td1" dans lequel on va travailler :

mkdir td1
cd td1

  • terminal : xterm, konsole

 Lancer une fenêtre "terminal" ou plusieurs si nécessaire :

xterm & ou konsole&

  • Apprentissage de l'utilisation de base du shell

Créer des fichiers vides avec la commande touch

 > touch aaa aab aac   abb abc acd bcd

à l'aide de ces fichiers apprenez à manipuler les commandes de base de gestion de fichiers :  cp, rm, mv, cd, mkdir, rmdir...

Structure des répertoires :   Chemin absolu, chemin relatif ,  / . .. ~ ~username

Utiliser chmod pour donner des permissions différentes à chacun de ces fichiers

Apprenez à utiliser les commandes du shell d'édition de la ligne de commande :

flèches du clavier
[^D] : Choix possibles (de fichiers ou de commandes)
[TAB] : "Completion" automatique si choix (ou tant que choix) unique
* et ?: "Expansion" des "Jockers"

  • Utilisation du man

man "commande"

  • Les processus
liste des processus : commande ps (voir man ps) 
ps -u sr01xxx

gestion des processus: Arrêt, Avant plan, Arrière plan
apprenez la différence entre "gedit fichier <return>"  et "gedit fichier &<return>"

    • A partir du répertoire racine (celui qui est à la base de l'arborescence ; il s'agit ici de ~), créez l'arborescence suivante : .

Les premiers programmes C

Ce TD est à réaliser dans le répertoire td1.
 

  • hello1.c

Avec gedit , créer un fichier "hello1.c" 
contenant le programme C suivant :

/* hello1.c*/

#include <stdio.h>



main ()

{



printf("hello world!");

printf("bye world!");



return 0;



}
Compiler ce code avec : 
gcc -o hello1 hello1.c
Lancer le programme hello1

faire une nouvelle version de votre programme (hello2.c) en ajoutant le caractère '\n' à la fin de chaque phrase.
recompiler et executer hello2.c

  • Code ascii: ex1a.c

copier , compiler et executer le programme ex1a.c suivant:

/* ex1a.c*/

#include <stdio.h>
